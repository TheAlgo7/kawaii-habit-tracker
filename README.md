# Kawaii Habit Tracker

A React habit tracker wrapped in a soft, character-driven experience where progress is reflected through a companion and evolving world. The project leans into emotional design rather than treating habits like a sterile checklist.

## Highlights

- Habit tracking through a playful companion-centered UI
- React app with a lightweight frontend architecture
- Installable PWA-style behavior for everyday use
- Fast iteration through a Vite development workflow

## Stack

- React
- Vite
- JavaScript
- CSS
- Static frontend deployment support

## Project structure

- `src/KawaiiHabit.jsx` - primary experience logic
- `src/App.jsx` and `src/main.jsx` - app shell and bootstrap
- `src/App.css` and `src/index.css` - styling layers
- `public/` - static assets
- `api/` - deployment-side helpers
- `vercel.json` - hosting configuration

## Local development

1. Install dependencies with `npm install`.
2. Run `npm run dev`.
3. Use `npm run build` for production output.
4. Use `npm run preview` to inspect the built app locally.

## Notes

- The current package is configured as `kawaii-habit-tracker`.
- The repo is strongest when treated as a personality-rich frontend product rather than a generic productivity tool.
